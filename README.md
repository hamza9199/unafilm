# UnaFilm

UnaFilm je web aplikacija za distribuciju filmova i novosti iz svijeta filma. Platforma omogućava korisnicima pregled filmova, trejlera, vijesti i drugih sadržaja, dok administratori imaju mogućnost upravljanja sadržajem putem kontrolne table.
